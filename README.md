## 下载、安装和测试
//  克隆项目 或者直接下载zip解压
git clone https://git.coding.net/ben302010/download-app.git

//从项目目录打开cmd  安装相关依赖
npm install

// 本地部署
npm start

浏览器输入127.0.0.1:1337


##项目的目录结构说明如下：

```
├── cafeDeliver                    //根目录
│   ├── widget                       //app文件目录
│         ├── css                            //css样式文件目录
│         ├── error
│         ├── html                              //（旧）app页面的html文件
│         ├── icon
│         ├── image                             //app图片文件
│         ├── launch
│         ├── newAppHtml                        //（新）app页面的html文件

│         ├── res
│         ├── script                            //（旧）app页面的js文件
│         ├── config.xml                        //apiCloud启动配置
│         ├── index.html                //App首页
│         └── new_login.xml              //登陆页
```
